# ⏱️ Simple Stopwatch

A clean, beginner-friendly stopwatch made with **HTML**, **CSS**, and **JavaScript**.  

## 🔥 Features
- Start, Stop, and Reset buttons 🟢🔴🔁
- Displays **hours, minutes, seconds**, and **centiseconds** (1/100 of a second)
- Smooth transitions and hover effects
